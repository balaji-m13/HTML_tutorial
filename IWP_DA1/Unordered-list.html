Unordered Lists in HTML

Unordered_list_in_html

Unordered lists are not ordered and are bulleted.

An unordered list starts with <ul> tag and ends with <ul/>. The list elements are entered using the <li> tag.
The following is a syntax\structure of an unordered list in HTML
<ul>
<li>...</li>
</ul>.

The following lines depict an unordered list in HTML.Unordered lists are marked with a bullet.

  • Messi
  • Zlatan
  • Pogba
  • Aubameyang

There are three types of bullets:disc,circle and square

The bullets can be specified using the type attribute of ul tag as follows:
<ul type="circle|square|disc">

  • disc
  • circle
  • square
