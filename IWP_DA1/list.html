Ordered Lists in HTML

Ordered_list_in_html

Ordered lists are ordered based on numbers or roman numerals

An ordered list starts with <ol> tag and ends with <ol/>. The list elements are entered using the <li> tag.
The following is a syntax\structure of an ordered list in HTML
<ol>
<li>...</li>
</ol>.

The following lines depict an ordered list in HTML.Ordered lists are marked with an indexing or numbering.

  1. Messi
  2. Zlatan
  3. Pogba
  4. Aubameyang

There are different types of ordering like

  1. Type '1', the list items will be numbered with numbers.
  2. Type 'A', the list items will be ordered with uppercase letters.
  3. Type 'a', the list items will be ordered with lowercase letters.
  4. Type 'I', the list items will be ordered with uppercase roman numbers.
  5. Type 'i', the list items will be ordered with lowercase roman numbers.
