Basic tags in HTML

The following are basic tags in HTML

html tag: It defines an HTML document


head Tag: Contains metadata/information for the document


title tag: Defines a title for the document


body tag: Defines the document's body


Heading tags(h1 to h6): Defines HTML headings


p tag: Defines a paragraph


br tag: Inserts a line break


hr tag: Defines a thematic change in the content


<!--...--> tag: Defines a comment
