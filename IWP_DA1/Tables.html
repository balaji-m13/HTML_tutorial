Tables in HTML

The HTML tables allow web authors to arrange data like text, images, links, other tables, etc. into rows and columns of cells. The HTML tables are created using the table tag in which the tr tag is used to create table rows and tag is used to create data cells. The elements under are regular and left aligned by default
