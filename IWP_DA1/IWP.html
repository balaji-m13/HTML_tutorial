Basic tags and formatting tags in HTML

Few basic and formatting tags in HTML are:

  • !DOCTYPE:A doctype or document type declaration is an instruction which tells the web browser about the markup language in which the current page is written. The Doctype is not an element or tag, it lets the browser know about the version of or standard of HTML or any other markup language that is being used in the document. A DOCTYPE declaration appears at the top of a web page before all other elements. According to the HTML specification or standards, every HTML document requires a document type declaration to ensure that the pages are displayed in the way they are intended to be displayed.
    For HTML documents, we give the DOCTYPE as html

  • html tag: This tag is used in all HTML programs. It indicates the begining of the html program. This is the tag that marks that the document should be interpreted as a HTML document.

  • head tag: The head of an HTML document is the part that is not displayed in the web browser when the page is loaded. It contains information such as the page title, links to CSS (if you choose to style your HTML content with CSS), links to custom favicons, and other metadata (data about the HTML, such as the author, and important keywords that describe the document.)
  • title tag: This contains the title of the webpage, it is the name that appears on the browser tab when we open the HTML document

  • body tag: This tag is used for defining the document's body.

  • hr tag: Used for drawing a horizontal line or a horizontal rule.
  • br tag: It is known as break tag.It is used for inserting a line break.
  • hr tag: Used for horizontal rule.
  • heading tag: HTML has 6 heading tags(h1 to h6) which define HTML headings. The size of the text decreases from h1 to h6.The following is an example of heading tags from h1 to h6.

Forms in HTML

Form tag is a new inclusion in HTML 5. It is a great method to create online forms. There are many different kinds of inputs like radiobutton, checkboxes etc. which are supported by the form tag. Along with forms often a label is used to indicate the name of the field for the convinience of the user. The information entered through the form is then accessed and used using JavaScript and PHP at the back end.

The following are some of the input types supported by HTML:

  1. Text: This allows us to get a text input often used for text fields such as username and password.
    A text box can be implemented by assigning the type attribute to "text" inside the input tag.
    Syntax is:<input type="text"/>

  2. Radio button: Radio button can be used in scenarios where the user has the choice of selecting only one of the options. It is often used in 'yes' or 'no' cases, gender selection etc.
    A radio button can be rendered by assigning the type attribute of the input tag to "radio".
    Syntax is: <input type="radio"/>

  3. Checkbox: Checkboxes are used in scenarios where the user can/has to select two or more of the given choices.Commonly used in check lists.
    It can be rendered by assigning the type attribute of input tag to "checkbox".
    Syntax is: <input type="checkbox"/>

  4. file: Files are generaaly used in forms where the user has to upload a file. Mostly used in sites where verification of one's photograph is required or one's resume is required.
    A file upload option can be included by assigning the type attribute of input tag to "file".
    Syntax is: <input type="file"/>

  5. email: Used for accepting emails via forms. It checks whether the @ symbol and the period '.' is present in the user entered string.
    Syntax is: <input type="email"/>

  6. submit and reset: These buttons are generally included in all form. The submit button is for submitting the form and recording our responses. The reset button is for clearing all our responses to start from scratch
    Syntax is:<input type="submit|reset"/>

Select tag:

It is used for creating a drop down list containing many options.The option tag is used for specifying the options.
Syntax is:<select> <option> ...</option> </select>

Links in HTML

A link is specified using HTML tag <a>. This tag is called anchor tag and anything between the opening <a> tag and the closing </a> tag becomes part of the link and a user can click that part to reach to the linked document. Following is the simple syntax to use <a> tag.

<a href="..." target="_blank|self|iframe">...</a>.

The <a> tag has two main attributes which are href and target. href is used to specify the source URL or the path of the webpage to which it should navigate to on clicking upon the link. The target attribute determines the window or the tab in which the link opens. If we set the target as _self then the link gets loaded in the same tab. If we give the target as _blank, the link gets loaded in a separate tab.If we give the target as iframe, then it opens in the corresponding iframe.

Tables in HTML

A table is an arrangement of data in rows and columns, or possibly in a more complex structure. Tables are widely used in communication, research, and data analysis. Tables are useful for various tasks such as presenting text information and numerical data. Tables can be used to compare two or more items in tabular form layout. Tables are used to create databases.
Tables are created using <table> tag. The rows are defined using <tr> tag.The table headers are defined using <th> tag.The table data is defined using <td> tag.
Cell spacing and cell padding can also be implemented. We can use colspan and rowspan to merge rows and columns.

Syntax is:
<table><tr> <th>...</th> <td> ... </td> </tr> </table>

Image and image map

HTML allows us to add images in webpages using <img> tag. We can set different sections of the image to navigate to different webpages by using imagemap.
The img tag must contain a src attribute which is mandatory. We specify the source of the webpage to be navigated inside the src attribute.
The following is the Syntax for img tag in HTML:
<img src=".."/>
Image map is also called client side mapping. We link sections of an image to create an image map.
In order to use image map, we use the attribut usemap which is assigned to the corresponding map Id and another attribute ismap for mapping.Apart from this, we use a map tag which contains the details of the areas of the image to be mapped for which area tag is used. Inside the area tag, we specify the type of shape and coordiantes of the image to be mapped.

The following is the syntax to implement image map:
<img src="..." usemap="#one" ismap/>
<map name="#one">
<area shape="poly|circle|rectangle" coords="..." href="..."/>
</map>

iframes in HTML

An iframe or inline frame is used to display external objects including other web pages within a web page. An iframe pretty much acts like a mini web browser within a web browser. Also, the content inside an iframe exists entirely independent from the surrounding elements.

The basic syntax for adding an iframe to a web page can be given with:
<iframe src="URL"></iframe>

The URL specified in the src attribute points to the location of an external object or a web page. We can also set the target of <a> tag to iframe so that it opens the link in the particular iframe.
